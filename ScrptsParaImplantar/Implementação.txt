 SEQUÊNCIA DE IMPLEMENTAÇÃO PASSO A PASSO
HORA 1 - CONFIGURAÇÃO BÁSICA
✅ Criar todos os scripts na pasta Scripts/

✅ Configurar hierarquia de objetos:

Jogador (nome: "Pleno", tag: "Player")

SpawnManager (objeto vazio)

ItemSpawner (objeto vazio)

LocaisSpawn (objetos vazios para spawn points)

HORA 2 - SISTEMA DE ÁUDIO
✅ Configurar SoundOnObject:

Anexar aos objetos que emitem som

Preencher arrays de áudio no Inspector

Verificar nomes dos GameObjects

✅ Configurar áudios do Jogador:

Arrastar clips para somTrombando e somMorrendo

HORA 3 - SPAWN SYSTEM
✅ Configurar SpawnManager:

Arrastar prefabs de ruídos (ruidoD, ruidoE, ruidoT)

Arrastar locais de spawn (localSpawnE, localSpawnD, localSpawnT)

✅ Configurar ItemSpawner:

Arrastar prefabs (moeda, obstaculo, passaro)

Arrastar locais de obstáculos (localSpawnC0, C1, C2)

Ajustar intervalos

HORA 4 - SISTEMA DE COLETA
✅ Configurar prefabs de coletáveis:

Moeda: Coletavel → Tipo = Moeda, configurar som

Pássaro: Coletavel → Tipo = Passaro, configurar som

✅ Testar sistema de coleta:

Clicar em moedas/pássaros deve tocar som e destruir

Verificar estatísticas no console

HORA 5 - CONTROLES E UI
✅ Configurar controles do jogador:

Anexar MoveJogador ao jogador

Criar botões UI e arrastar para o script

Implementar o script DesativarBotoesMobile.cs em cada botão

Ajustar limites de movimento

✅ Configurar colisões:

Adicionar Colisor nos objetos que causam dano

Verificar Colliders como triggers

HORA 6 - TESTES FINAIS
✅ Testar fluxo completo:

Movimento do jogador

Spawn de itens e ruídos

Sistema de coleta

Sistema de dano e vida

Sons em todos os eventos

✅ Ajustes finais:

Balancear velocidades

Ajustar intervalos de spawn

Polir sons e feedbacks visuais
